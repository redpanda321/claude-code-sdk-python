# Security Policy

## Supported versions

| Version     | Supported |
|-------------|-----------|
| 0.1.x (Alpha) | ✅ |
| < 0.1       | ❌ (not released) |

## Reporting a vulnerability

Please **do not** open a public GitHub issue for security bugs.

Report vulnerabilities privately via one of:

1. **Email**: redpanda321@hotmail.com -- subject `[SECURITY] claude-code-sdk`.
2. **GitHub private advisory**: use the
   [Report a vulnerability](https://github.com/redpanda321/claude-code-sdk-python/security/advisories/new)
   form.

Include: affected version / commit, reproduction, and impact assessment.

## Response expectations

- **Acknowledgement**: within **7 days**.
- **Initial triage + severity**: within **14 days**.
- **Fix or mitigation plan**: communicated to the reporter; disclosure is
  coordinated.

Thanks for helping keep the Python ecosystem secure.
